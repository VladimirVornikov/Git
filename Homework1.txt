1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
    -Установить Git Bash(Если операционная система Windows)
    -Скопировать SSH-код
    -Через терминал вести команду git clone [SSH-код]
2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
    -cd /path/workingFolder
    -git init
    -git add .
    -git commit -m "[Описание]"
    -git push
3. Как получить на локальный компьютер изменения с github.com
    -git pull
4. Какую информацию нам дают команда :
    -git status - отображает, текущее состояние вашего репозитория(измененные/неизмененные файлы)
    -git log - отображает, историю коммитов репозитория
    =git remote -v  - выводит список удаленных репозиториев, связанных с вашим локальным репозиторием.
